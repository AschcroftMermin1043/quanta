⚙ One-time Setup

Follow these steps once. After this, every upload instantly appears for all visitors — stored in your own Google Sheet.

1

Create a Google Sheet

Go to sheets.new to create a new spreadsheet. In Row 1, add these exact column headers (copy-paste to be safe):

id title desc subject type difficulty tags link date
2

Open Apps Script

In your sheet: Extensions → Apps Script. Delete all existing code, paste the script below, then click Save (Ctrl+S).

const SHEET_NAME = 'Sheet1'; const PIN = '1234'; // ← change this to your own secret PIN function doGet() { const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(SHEET_NAME); const data = sheet.getDataRange().getValues(); const headers = data[0]; const rows = data.slice(1).map(row => { const obj = {}; headers.forEach((h, i) => obj[h] = row[i]); return obj; }); return ContentService .createTextOutput(JSON.stringify(rows)) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const body = JSON.parse(e.postData.contents); if (body.pin !== PIN) { return ContentService .createTextOutput(JSON.stringify({ error: 'bad pin' })) .setMimeType(ContentService.MimeType.JSON); } const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(SHEET_NAME); sheet.appendRow([ body.id, body.title, body.desc, body.subject, body.type, body.difficulty, body.tags, body.link, body.date ]); return ContentService .createTextOutput(JSON.stringify({ ok: true })) .setMimeType(ContentService.MimeType.JSON); }
3

Deploy as Web App

In Apps Script: click Deploy → New deployment → gear icon → select type Web app → set "Execute as" to Me and "Who has access" to Anyone → click Deploy → Authorize when prompted → copy the Web App URL it shows you.

4

Paste your URL & set your PIN

Paste the URL from step 3 below, set the same PIN you used in the script, and save.
